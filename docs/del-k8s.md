## Remover a infraestrutura do K8s

1. Para o Minikube.

```bash
minikube stop
```

<p align="center">
  <img src="../docs/images/img18.png" alt="Img18" />
</p>

2. Excluir o Minikube.

```bash
minikube delete
```
